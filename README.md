# Smart Medicine Reminder and Dose Tracking System

## Overview

This project is designed to remind users to take their medicine on time using an LCD display, LED, and buzzer. The user can confirm medicine intake using a push button, and the system keeps track of the dose count.

## Components Used

* Arduino UNO
* LCD 16x2 I2C Display
* Push Button
* LED
* Buzzer
* 220Ω Resistor

## Features

* Medicine Reminder Alert
* LCD Notification Display
* User Confirmation Button
* Dose Tracking Counter
* Missed Dose Detection

## Working

1. LCD displays reminder message.
2. LED and buzzer alert the user.
3. User presses the button after taking medicine.
4. Dose count increases and confirmation message is displayed.
5. If no response is received within the given time, a missed dose alert is shown.

## Applications

* Healthcare Monitoring
* Elderly Care
* Medication Management
* Patient Assistance
